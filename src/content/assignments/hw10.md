## 作业目标

学习Vue.js 3基础，掌握插值表达式、指令、事件处理和计算属性。

## 功能要求

1. **待办事项列表**：
   - 使用`v-for`渲染列表
   - 使用`v-model`双向绑定输入框
   - 使用`v-on`/@添加、删除、切换完成状态

2. **条件渲染**：
   - 使用`v-if/v-else`显示空状态提示
   - 使用`v-show`控制元素显示/隐藏

3. **样式绑定**：
   - 使用`:class`绑定完成状态样式（删除线）
   - 使用`:style`动态调整样式

4. **统计功能**：
   - 计算属性统计总任务数、已完成数、未完成数

## 技术要求

- Vue 3 CDN引入
- `createApp`创建应用
- `data()`返回响应式数据
- `methods`定义方法
- `computed`定义计算属性

## 提示

```javascript
const { createApp } = Vue;

createApp({
  data() {
    return {
      todos: [],
      newTodo: ''
    }
  },
  computed: {
    completedCount() {
      return this.todos.filter(t => t.done).length;
    }
  },
  methods: {
    addTodo() { /* ... */ },
    removeTodo(index) { /* ... */ }
  }
}).mount('#app');
```
