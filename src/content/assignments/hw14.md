## 作业目标

学习Vue Router实现单页应用(SPA)路由管理，掌握路由配置、导航和传参。

## 功能要求

1. **路由配置**：
   - 首页(`/`)：显示欢迎信息
   - 产品列表(`/products`)：显示所有产品
   - 产品详情(`/products/:id`)：显示单个产品详情
   - 购物车(`/cart`)：显示购物车内容
   - 404页面：未匹配路由

2. **导航菜单**：
   - 使用`<router-link>`创建导航
   - 当前路由高亮显示
   - 支持编程式导航

3. **动态路由**：
   - 产品详情页接收id参数
   - 根据id获取并显示对应产品
   - 返回按钮返回列表

4. **路由守卫**（可选）：
   - 进入购物车前检查登录状态
   - 未登录重定向到登录页

## 技术要求

- Vue Router 4 CDN引入
- `createRouter`创建路由实例
- `createWebHashHistory`使用hash模式
- `useRoute`获取当前路由信息
- `useRouter`进行编程式导航

## 提示

```javascript
const { createRouter, createWebHashHistory } = VueRouter;

const routes = [
  { path: '/', component: Home },
  { path: '/products', component: ProductList },
  { path: '/products/:id', component: ProductDetail },
  { path: '/cart', component: Cart },
  { path: '/:pathMatch(.*)*', component: NotFound }
];

const router = createRouter({
  history: createWebHashHistory(),
  routes
});

app.use(router);
```
