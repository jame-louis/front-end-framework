## 作业目标

创建一个键盘按键显示器，实时显示用户按下的按键信息，练习键盘事件的处理。

## 功能要求

1. **按键显示**：实时显示按下的键名（如"Enter"、"Shift"）
2. **按键代码**：显示按键的keyCode或code值
3. **组合键检测**：检测并显示Ctrl、Shift、Alt等组合键
4. **历史记录**：显示最近按下的10个按键历史
5. **视觉反馈**：按键时有视觉动画效果

## 技术要求

- 使用`keydown`和`keyup`事件
- 使用`event.key`、`event.code`、`event.keyCode`获取按键信息
- 使用`event.ctrlKey`、`event.shiftKey`等检测修饰键

## 提示

```javascript
document.addEventListener('keydown', (event) => {
  console.log(event.key);      // 键名，如 "Enter"
  console.log(event.code);     // 物理位置，如 "KeyA"
  console.log(event.keyCode);  // 数字代码（已废弃但兼容性高）
  
  // 检测组合键
  if (event.ctrlKey && event.key === 's') {
    event.preventDefault();  // 阻止默认保存行为
    console.log('Ctrl+S 被按下');
  }
});
```
