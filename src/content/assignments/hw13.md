## 作业目标

理解Vue组件生命周期，掌握异步数据获取和loading状态管理。

## 功能要求

1. **异步数据获取**：
   - 在`onMounted`钩子中获取数据
   - 使用fetch API获取远程数据
   - 处理loading状态和错误状态

2. **用户列表页面**：
   - 显示从API获取的用户列表
   - 显示loading动画
   - 错误时显示错误信息
   - 支持刷新数据

3. **分页功能**：
   - 实现前端分页
   - 显示页码导航
   - 切换页码时保留已加载数据

4. **详情弹窗**：
   - 点击用户显示详情
   - 异步获取用户详情数据
   - 显示加载状态

## 技术要求

- `onMounted`组件挂载后执行
- `fetch` API获取数据
- `async/await`处理异步
- `v-if/v-else`控制loading/错误/数据状态

## 提示

```javascript
const { createApp, ref, onMounted } = Vue;

createApp({
  setup() {
    const users = ref([]);
    const loading = ref(false);
    const error = ref(null);
    
    const fetchUsers = async () => {
      loading.value = true;
      error.value = null;
      try {
        const res = await fetch('https://api.example.com/users');
        users.value = await res.json();
      } catch (err) {
        error.value = err.message;
      } finally {
        loading.value = false;
      }
    };
    
    onMounted(fetchUsers);
    
    return { users, loading, error, fetchUsers };
  }
}).mount('#app');
```
