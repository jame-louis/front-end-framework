## 作业目标

练习localStorage、sessionStorage的使用，以及JSON序列化和反序列化操作。

## 功能要求

1. **用户偏好设置存储**：
   - 主题选择（亮色/暗色）
   - 字体大小调整
   - 用户昵称
   - 使用localStorage持久化

2. **表单草稿自动保存**：
   - 用户在表单中输入时自动保存到localStorage
   - 页面刷新后恢复草稿内容
   - 敏感字段（密码）不保存
   - 成功提交后清除草稿

3. **访问统计**：
   - 记录页面访问次数
   - 显示首次访问时间
   - 显示最近访问时间
   - 使用sessionStorage和localStorage结合

## 技术要求

- `localStorage.setItem()` / `getItem()`
- `JSON.stringify()` 序列化对象
- `JSON.parse()` 反序列化
- `sessionStorage` 会话级存储

## 提示

```javascript
// 存储对象
const user = { name: '张三', theme: 'dark' };
localStorage.setItem('userPrefs', JSON.stringify(user));

// 读取对象
const saved = JSON.parse(localStorage.getItem('userPrefs'));

// 自动保存表单
form.addEventListener('input', () => {
  const data = Object.fromEntries(new FormData(form));
  delete data.password; // 不保存敏感字段
  localStorage.setItem('formDraft', JSON.stringify(data));
});
```
