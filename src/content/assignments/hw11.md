## 作业目标

深入理解Vue计算属性(computed)和侦听器(watch)，实现复杂的业务逻辑处理。

## 功能要求

1. **图书借阅车**：
   - 显示图书列表，可添加到借阅车
   - 使用计算属性统计借阅车：
     - 商品总数量
     - 原始总价
     - 折扣后总价（满100打9折，满200打8折）
     - 节省金额

2. **搜索过滤**：
   - 使用计算属性实现实时搜索过滤
   - 根据书名、作者、价格区间过滤

3. **排序功能**：
   - 按价格、评分、出版日期排序
   - 支持升序/降序切换

4. **数据持久化**：
   - 使用watch监听借阅车变化
   - 自动保存到localStorage
   - 页面刷新后恢复借阅车数据

## 技术要求

- `computed`缓存计算结果
- `watch`监听数据变化执行副作用
- `deep: true`深度监听对象变化
- `immediate: true`立即执行回调

## 提示

```javascript
computed: {
  // 缓存计算结果
  discountPrice() {
    const total = this.cart.reduce((sum, item) => sum + item.price, 0);
    if (total >= 200) return total * 0.8;
    if (total >= 100) return total * 0.9;
    return total;
  }
},
watch: {
  // 监听借阅车变化
  cart: {
    deep: true,
    handler(newVal) {
      localStorage.setItem('cart', JSON.stringify(newVal));
    }
  }
}
```
