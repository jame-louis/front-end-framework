## 作业目标

创建一个功能完整的图片轮播器组件，练习DOM操作、事件处理和定时器的综合应用。

## 功能要求

1. **自动轮播**：图片每隔3秒自动切换
2. **手动控制**：
   - 上一张/下一张按钮
   - 指示器（小圆点）点击跳转
3. **循环播放**：到最后一张后回到第一张
4. **暂停功能**：鼠标悬停时暂停自动播放
5. **过渡效果**：图片切换时有淡入淡出或滑动效果

## 技术要求

- 使用数组存储图片数据
- 使用`setInterval`实现自动轮播
- 使用CSS transition实现过渡效果
- 使用事件委托优化指示器点击

## 提示

```javascript
// 图片数据数组
const slides = [
  { url: 'image1.jpg', title: '图片1' },
  { url: 'image2.jpg', title: '图片2' },
  // ...
];

// 当前索引
let currentIndex = 0;

// 定时器控制
let timer = setInterval(nextSlide, 3000);

// 鼠标悬停暂停
container.addEventListener('mouseenter', () => clearInterval(timer));
container.addEventListener('mouseleave', () => timer = setInterval(nextSlide, 3000));
```
