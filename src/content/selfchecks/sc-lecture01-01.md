question: Web的三个时代分别是什么，各自有什么核心特征？
answer: Web 1.0（1990-2004）是只读的静态门户时代；Web 2.0（2004至今）是读写的社交互动时代；Web 3.0（正在兴起）是读写+拥有的去中心化时代。
explanation: |
  Web 1.0用户只是内容消费者，数据归属网站；Web 2.0用户成为内容创造者，但数据归属平台；Web 3.0用户拥有自己的数据，通过区块链技术实现。

  对比表：
  - Web 1.0：只读、消费者、网站管数据、门户广告、HTML/CSS
  - Web 2.0：读写、创造者、平台管数据、平台经济、Ajax/移动
  - Web 3.0：读写+拥有、所有者、自己管数据、代币经济、区块链
