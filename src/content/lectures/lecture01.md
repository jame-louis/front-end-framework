## 一、课程概述

### 1.1 课程定位

本课程旨在帮助学生掌握现代前端开发的核心技术，培养独立构建 Web 应用的能力。

### 1.2 学习路径

```
基础入门 → 核心概念 → 框架应用 → 项目实战
```

### 2.1 必备工具

| 工具 | 用途 | 下载地址 |


## 三、第一个示例

```html
<!DOCTYPE html>
<html lang="zh-CN">

  <meta charset="UTF-8">

</head>

  <h1>Hello, World!</h1>
</body>
</html>
```
