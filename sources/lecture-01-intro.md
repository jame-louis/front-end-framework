## Learning Objectives

By the end of this lecture, you will be able to:

- Understand the course structure and assessment criteria
- Set up your development environment
- Complete your first exercise

## Course Overview

<!-- expand -->
- This course provides a comprehensive introduction to modern web development using Astro and React. We will cover everything from basic component architecture to advanced state management patterns, with a focus on building production-ready applications.
<!-- endexpand -->

### What You Will Learn

- Component-based architecture
- Static site generation
- Client-side interactivity
- Deployment strategies

### Assessment Structure

- Weekly assignments (40%)
- Midterm project (30%)
- Final project (30%)

## Development Environment

### Required Tools

- Node.js 20 or higher
- Git
- VS Code (recommended)

### Optional but Recommended

- GitHub account
- Vercel or Netlify account for deployment

## Summary

This course combines theoretical knowledge with hands-on practice. Complete the environment setup assignment before the next lecture.
