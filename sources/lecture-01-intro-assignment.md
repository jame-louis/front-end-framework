## Description

Set up your development environment and verify it works by running the starter template.

## Tasks

- [ ] Install Node.js 20+ (verify with `node --version`)
- [ ] Install Git (verify with `git --version`)
- [ ] Create a GitHub account if you don't have one
- [ ] Clone the course starter template
- [ ] Install dependencies with `npm install`
- [ ] Start the dev server with `npm run dev`
- [ ] Verify the site loads at `http://localhost:4321`

## Deliverables

1. Screenshot of browser showing the running application
2. Link to your forked GitHub repository

| Criteria | Points |

| Node.js correctly installed | 2 |
