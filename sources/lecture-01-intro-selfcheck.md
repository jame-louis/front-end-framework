## Self Check

Q: What percentage of the final grade do weekly assignments contribute?
A: 40%
Explanation: Weekly assignments account for 40% of the final grade, the midterm project is 30%, and the final project is also 30%.

Q: What is the minimum required Node.js version for this course?
A: Node.js 20
Explanation: The course requires Node.js 20 or higher to ensure compatibility with all the tools and dependencies we will be using.

Q: What port does the Astro dev server use by default?
A: Port 4321
Explanation: Astro's default development server runs on port 4321, which you can access at http://localhost:4321 in your browser.

Q: Which IDE is recommended for this course?
A: VS Code
Explanation: While you can use any editor, VS Code is recommended because it has excellent support for Astro and the extensions we will use.
